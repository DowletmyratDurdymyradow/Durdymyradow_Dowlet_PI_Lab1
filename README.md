# Отчёт по выполненным действиям

## 1. Создание репозитория на GitHub

Я выполнил следующие шаги для создания нового репозитория на GitHub:

1. Открыл сайт [github.com](https://github.com) и вошёл в свой аккаунт.
2. Нажал кнопку **New repository** (Создать репозиторий).
3. Указал название репозитория и при необходимости добавил краткое описание.
4. Выбрал тип репозитория: **Public** или **Private**.
5. Нажал кнопку **Create repository**.
6. Скопировал команды, предложенные GitHub для подключения локального проекта.
7. В терминале выполнил базовые команды:

```bash
git init
git add .
git commit -m "Первый коммит"
git branch -M main
git remote add origin <ссылка_на_репозиторий>
git push -u origin main
```

В результате локальный проект был успешно загружен в созданный репозиторий на GitHub.

## 2. Установка VirtualBox на Windows

Для установки VirtualBox на Windows я сделал следующее:

1. Перешёл на официальный сайт VirtualBox: [https://www.virtualbox.org](https://www.virtualbox.org).
2. Открыл раздел **Downloads**.
3. Скачал установщик **VirtualBox for Windows hosts**.
4. Запустил скачанный `.exe`-файл от имени администратора.
5. В мастере установки оставил стандартные параметры (или выбрал нужные компоненты вручную).
6. Подтвердил установку сетевых драйверов VirtualBox, когда появилось системное предупреждение.
7. Дождался завершения установки и нажал **Finish**.
8. После установки запустил VirtualBox и проверил, что программа открывается без ошибок.

## 3. Итог

В ходе работы я:

- создал репозиторий на GitHub;
- подключил локальный проект к удалённому репозиторию;
- установил VirtualBox на Windows и проверил его запуск.

Все задачи выполнены успешно.
